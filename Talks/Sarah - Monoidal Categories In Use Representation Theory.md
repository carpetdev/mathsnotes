# Motivation
- Rep theory studies a structure by studying structure preserving maps $R\colon S\to T$ to a better understood target
- Motivating example (MR21), $S$ is braid category with restricted $T$ matrix category
    - Trivial matrix category not useful
    - We go for a monoidal category $\cat{Mat}^\otimes$
    - $R(1) = N \in \mathbb{N}$
    - Start with $\cat{Mat}^N$ a monoidal subcategory of $\cat{Mat}^\otimes$ generated by object $N$
    - We want a smaller target and we take $\cat{Match}^N$
        - $\cat{Match}_\text{diag} \subseteq \cat{Match}^N \subseteq \cat{Mat}^N \subseteq \cat{Mat}^\otimes$ 
    - Our case $B=\langle \sigma,\sigma^{-1} \rangle$, $\sigma\in B(2,2)$, $F\colon B\to \cat{Mat},\ 1\mapsto N$, so $F(\sigma) \in \cat{Mat}^N(2,2)$, so $N$ free parameters -- too many so we must reduce

# Definitions
- Defined $\cat{Mat}^N$
- Charge conserving if $\langle w | R | v \rangle \neq 0$ implies $w$ is a permutation of $v$. Linear strict monoidal subcategory under kronecker product
- Additive charge conserving if the same implies $\sum w_i = \sum v_i$. Also subcategory
- $\cat{Match}^N(n,n)$ and additive version are semisimple algebras
- Write $\Lambda^N_n$ for the set of $N$-part weak compositions of $n$. That is, $\Lambda^N_n = \{m\in \mathbb{N}^N_0 : m.(1,1,\ldots,1)=n\}$
- Multinomial coefficients generalise binomial coefficients
- s-binomial coefficients generalise too
