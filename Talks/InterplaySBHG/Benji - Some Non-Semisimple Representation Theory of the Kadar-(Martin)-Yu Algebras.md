- TL Algebra:
    - $\mathbb{K}$ commutative ring, $\alpha\in\mathbb{K}$, $n\in \mathbb{N}$
    - Diagrams connecting ordered points with non-crossing condition (maximally restrictive)
    - Multiply by stacking, remove loops with $\alpha$
- Minimally restrictive non-crossing condition (i.e. allow crossings) is the Brauer Algebra.
- $\dim(\operatorname{TL}_n)/\dim(\operatorname{Br}_n)\to \infty$
- KMY Algebra:
    - "Intermediate restriction"
    - Pick $l\in\set{-1,0,1,\dots}$
    - $J_{l,n}(\alpha) = \mathbb{K}\set{\text{pairings}\in \operatorname{Br}_n(\alpha)\text{ with "height"}\leq l}$
    - $\operatorname{TL}_n=J_{-1,n}\subseteq J_{0,n}\subseteq J_{1,n}\subseteq\dots\subseteq J_{n-1,n}=J_{\infty,n}=\operatorname{Br}_m$
    - Thm (KMY '19): $J_{l,n}$ is a subalgebra of $\operatorname{Br}_n$ for all $l$
    - Thm (Alraddadi-Parker '24): $J_{l,n}$ is generated by some things (TL + some crossings restricted based on $l$)
- Rep Theory:
    - Thm (AP): For $l$ fixed, the algebras $J_{l,n}$ satisfy Towers of Recollunet framework (Cox-M-P-Xi)
    - We can localise and globalise by tensoring modules with diagrams between $n$ and $n+2$ point. $LG = \id$ functor but $GL$ is not
    - CMPX framework says $J_{l,n}$ is semisimple if and only if there are no maps $\Delta^{(n)}_{(n,\lambda)}\longleftrightarrow \Delta^{(n)}_{(n-2,\mu)}$
        - Thm (M-Martin): There exists a Chebyshev series of polynomials giving such a map
