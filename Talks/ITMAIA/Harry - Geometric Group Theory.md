- Novikov-Boone: There is a finitely presented group such that it is undecidable if a given word is the identity (word problem not always solvable)
- Adian-Rabin: Let $P$ be a property of groups such that there exists a finitely presented group with $P$, and there is a finitely presented group which is not a subgroup of any $P$ group. It is undecidable whether a given presentation has $P$.
- Thus, we want to use geometry to help

- $G$ acts freely and transitively on $\operatorname{Cay}(G,S)$ ($S$ generates $G$)
- Fundamental group of genus 2 surface ${} \pi_1(S_2) {}$ is $\langle a,b,c,d \mid [a,b][c,d] \rangle$ has Cayley graph a bunch of octagons in the hyperbolic space $\mathbb{H}^2$
- Dehn's algorithm solves the word problem for $\pi_1(S_g)$
    - Claim: If $w$ represents the identity then there is a boundary octagon (for $g=2$) that $w$ traverses $\geq5$ edges of (build tiling picture up in layers to see this)
    - Make a shorter word by replacing with the $\leq3$ other edges
    - Check boundary octagon by just looking for certain subwords

- A geodesic metric space is **$\delta$-hyperbolic** if $\exists\delta\geq0$ such that for every geodesic triangle $\gamma_1\gamma_2\gamma_3$, we have $\gamma_1\subseteq$ $\delta$-neighbourhood of $\gamma_2\cup\gamma_3$
    - Examples
        - Trees are 0-hyperbolic
        - Bounded spaces
        - Hyperbolic space $\mathbb{H}^2$
        - Wedge sums of hyperbolic spaces
    - Non-Examples
        - $\mathbb{R}^2$
        - $\operatorname{Cay}(\mathbb{Z}^2,\{a,b\})$
- Fact: $\forall\delta\ \exists k_1,k_2$ such that if $X$ is $\delta$-hyperbolic then any path joining $x,y\in S(p,r)$ (circle centre $p$ radius $r$) has length $\geq \exp{k_1d(x,y)}-k_2$

- A finitely generated group is **hyperbolic** if it has a hyperbolic Cayley graph
    - Examples
        - Cayley graphs
        - Finite groups
        - $\pi_1(S_g)$
    - Non-Examples:
        - $\mathbb{Z}^2$ and all supergroups

- A **$q$-quasigeodesic** in a metric space is a path $\gamma$ such that $\frac{1}{q}|s-t|-q\leq d(\gamma(s),\gamma(t)) \leq q|s-t|+q$
    - E.g. Other two sides of an equilateral triangle in $\mathbb{R}^2$
- Morse lemma:  $\forall\delta,q\ \exists M$ such that if $X$ is $\delta$-hyperbolic, $\alpha\subseteq X$ is a geodesic, and $\gamma\subseteq X$ is a $q$-quasigeodesic with the same endpoints as $\alpha$, then $\alpha\subseteq \mathcal{N}_M(\gamma)$ and $\gamma\subseteq\mathcal{N}_M(\alpha)$. Proof uses Fact above
- Fact: If $\gamma\subseteq \operatorname{Cay}(G,S)$ is a geodesic, then $\gamma$ is a quasigeodesic in $\operatorname{Cay}(G,S')$
- Corollary: If one Cayley graph is hyperbolic then so are all others
    - Proof: Take a geodesic triangle in $\operatorname{Cay}(G,S)$. It is a quasigeodesic triangle in $\operatorname{Cay}(G,S')$. By Morse Lemma, it's closed to a geodesic triangle which is thin by hyperbolicitiy. Hence original triangle is thin. $\blacksquare$

- **Hausdorff distance** between $A,B\subseteq X$ is $\inf\set{r\mid A\subseteq \mathcal{N}_r(B),\ B\subseteq \mathcal{N}_r(A)}$
- Let $G$ hyperbolic group with Cayley graph $\Gamma$. The **boundary** ${} \partial G {}$ of $G$ is $\set{\text{geodesic rays based at }1\in\Gamma}/\text{finite Hausdorff distance}$ with topology generated by the $V(\gamma\in\partial G,r>0) = \set{\gamma' \mid d(\gamma(t),\gamma'(t) < 5\delta\ \forall t \leq r)}$
    - Examples
        - $G$ finite $\iff \partial G = \emptyset$ 
        - $G$ free $\implies\partial G = \partial(\text{tree}) \cong$ Cantor set
            - converse is $\exists$ a finite index subgroup that is free -- topological dimension 0 and add 1 to get cohomological dimension of group
        - $\partial \pi_1(S_g) \cong S^1$, $g\geq 2$
            - converse is true but hard
        - ${} \partial\pi_1(\text{closed hyperbolic }n\text{-manifold})\cong S^{n-1} {}$
            - converse not true for $n\geq 3$, but Cannon conjecture is that $\partial G\cong S^2$ means $G$ has a finite index subgroup  that is $\pi_1(M^3)$
- Lemma: Two Cayley graphs give the same boundary
    - Proof: Geodesics in one are quasigeodesic in the other. Morse lemma implies this is close to another geodesic. Taking this as the image gives an injective (by infinite Hausdorff distance argument) map $f$ one way (vice versa gives bijection). Let $\gamma_1\in\partial\Gamma_1$, $\gamma_2\in V(\gamma_1,r)$. Then in $\Gamma_2$ the distance is $\leq 5\delta q + q$ ($q$ is quasi number). By thin triangles, $f(\gamma_2) \in V(f(\gamma_2), r-\text{small amount})$, thus $f$ is open (vice versa gives homeomorphism).

- We can extend action of $G$ on $\Gamma$ to an action on $\partial G$ by homeomorphisms: Given $\gamma\in \partial G$, $g\gamma$ is a geodesic ray starting at $g$. By thin triangles, there is a corresponding point in $\partial G$ which is the image of $\gamma$
- Individual elements of $G$ act nicely on $\partial G$ (north-south dynamics): If $g\in G$ has infinite order then $\exists g_+,g_-\in\partial G$ such that for every open neighbourhood $A\subseteq \partial G$ of $g_+$ and every closed set $B$ in $\partial G$ avoiding $g_-$ $\exists N$ such that $g^n(B)\subseteq A\ \forall n\geq N$
    - Get $g_+,g_-$ from $\langle g \rangle$ which is a quasigeodesic so endpoints are different
- Ping-pong lemma: Suppose $G$ is hyperbolic and let $g,h\in G$ be infinite order elements such that $g_\pm, h_\pm$ are all distinct. Then $\exists N$ such that $\langle g^n,h^n \rangle \leq G$ is free $\forall n \geq N$
    - Proof: Let $A_\pm$ be neighbourhood of $g_\pm$ and same for $B$ and $h$ (boundary is compact Hausdorff so these can be disjoint). North-south dynamics gives $N$ such that if $n\geq N$, then $g^n(\partial G \setminus A_-) \subseteq A_+$ etc. Take a nonempty reduced word $w$ in $g^n,h^n$. If $w=g^nw'$ then $w'$ doesn't start with $g^{-1}$, so for $x\in\partial G\setminus\set{g_\pm,h_\pm}$, $wx\in A+$ and $w$ doesn't represent $1\in G$. $\blacksquare$
- Corollary (Tits' alternative): If $G$ is hyperbolic then either $G$ has cyclic subgroup of finite index or $F_2\leq G$
