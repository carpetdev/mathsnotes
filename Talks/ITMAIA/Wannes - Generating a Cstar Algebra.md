- A **quantum cluster algebra** is an algebra with a distinguished set of "positive" generators (cluster variables)
    - To do representation theory we want to make use of positivity
- **Positive representations** view the cluster variables as positive (unbounded) operators on some Hilbert space

- Unital $\mathbb{C}[X,Y]$ generated by $\mathbb{C}[X]$ and $\mathbb{C}[Y]$
- Non-unital $(XY)$ "generated" by $(X)$ and $(Y)$ (product of ideals)
    - Note $x,y\notin(XY)$

- $C_b(\mathbb{R}^2)$ continuous bounded functions $\mathbb{R}^2\to\mathbb{C}$ is unital and generated by $C_b(X)$ and $C_b(Y)$ where e.g. $X(x,y) = x$ and $C_b(X) = \set{g(X) \mid g\in C_b(\mathbb{R})}$
- $C_0(\mathbb{R}^2)$ continuous functions vanishing at infinity (e.g. $\exp{-{x^2+y^2}}$) is still "generated" by $C_0(X)$ and $C_0(Y)$ (closure of span of product)
    - Note $C_0(X) \subsetneq C_0(\mathbb{R}^2)$ but $C_0{X}\subseteq C_b(\mathbb{R}^2))$

## Categorical approach
- $F\colon \cat{Alg}\to \cat{Set}$, $F(A) = \set{(f,g) \mid f\colon \mathbb{C}[X]\to A,\ g\colon \mathbb{C}[Y]\to A}$. Then $F\cong \hom(\mathbb{C}[X,Y], -)$ is representable.
- $F\colon C^*\text{-}\cat{Alg}\to \cat{Set}$. What is $F(A)$? Not $F(A) = \newcommand{\mor}{\operatorname{Mor}}\mor(\set{C_0(X), C_(Y)}, A)$ (too big?)
    - Example of such a morphism is $\pi\colon \langle C_0(X),C_0(Y) \rangle\to A$, such that $[\pi(C_0(X))A] = A$, $[\pi(C_0(Y))A] = A$ non-degenerate 
- If $\mor(\set{A_1,\dots}, -)$ is representable by $C$ a $C^*$-algebra, then the $A_i$ **strongly generate** $C$

- F.G: There exists some Hilbert space $\mathcal{H}$ on which we can implement the cluster variables as positive unboudned operators
    - $C_0(A_i) \subseteq B(\mathcal{H})$ where $A_i$ are cluster variables
- A **positive representation** of the quantum cluster algebra is a map $\pi\colon \langle \set{C_0(A_i)} \rangle\to B(\mathcal{G})$ such that $[\pi(C_0(A_i))\mathcal{G}] = \mathcal{G}$. If $\mor(\set{C_0(A_i)}, -)$ is representable by some $C$ we call $C$ the quantum cluster $C^*$-algebra associated to the QCA
    - This is shown to exist in rank 0 and 1
